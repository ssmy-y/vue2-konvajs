# vue2-konvajs

#### 介绍
vue2项目中使用konvajs完成文字与矩形编辑及设置

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Run your unit tests
```
npm run test:unit
```

### Lints and fixes files
```
npm run lint
```
